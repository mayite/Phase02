### task04
实现用户注册功能
思路：
    用户输入用户名、密码
    将用户输入的内容按照固定的格式，比如：albert:123，存入文件
    可以往一个文件中重复注册新的用户名和密码
    附加：
       1、对输入的用户名进行合法性监测，不能以数字开头，且如果输入的用户名已存在于文件中则要求用户重新输入
       2、对输入的密码进行合法性监测，密码的长度至少6位，并且不能包含特殊字符*&$
